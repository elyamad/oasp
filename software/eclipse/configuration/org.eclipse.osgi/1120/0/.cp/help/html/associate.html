Associate a project with SonarQube

When started, the wizard will contact all referenced SonarQube servers and try to find the better match based on your Eclipse project name.

When no match is found or when the current choice is not what you expect you can click in the "SonarQube project" column. This is a text field with autocompletion. Like in SonarQube server search engine it works when you have typed at least 3 characters.
